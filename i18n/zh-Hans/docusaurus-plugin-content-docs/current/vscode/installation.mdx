# 安装

因为recorder使用了vscode的相当高的权限, 例如获取terminal的输入的命令以及命令的输出.

所以目前无法在微软的插件商店上架, 目前在open vsx中可以下载.

[下载地址](https://open-vsx.org/extension/TokenRollAI/recorder)
或者使用Github的[release](https://github.com/TokenRollAI/recorder/releases/latest)

如果您使用的是cursor/windsurf/trae, 默认使用的就是open-vsx

**可以直接在vscode的插件市场中搜索`recorder`进行安装.**

如果无法搜索到, 可以通过手动的方式实现安装.
