# Quick Start

Let's learn about **recorder in less than 5 minutes**.

This guide will walk you through the process of setting up and using the recorder for both VS Code and JetBrains IDEs. Please follow the steps in the sidebar to get started.

Here are the basic steps:
1. Install uv
2. Configure SPEC-mcp
3. Install the recorder plugin for your IDE
4. Click `Start Recording` at the bottom of your IDE to begin
